Masalu
======

> “Masalu! Pavavaljung. (Pavavalung)”

<dfn>Masalu</dfn> 意指台灣原住民排灣族語的「你好。」

這支 Node.js 寫成的小程式能夠協助你將 Hackpad 輸出的雜亂格式整理成終端機或其他純文字環境（如 [PTT](telnet://ptt.cc)、[PTT2](telnet://ptt2.cc)）可閱讀之格式。
